INFORMATION CAPTURE AND STORAGE

Psychologists say that for each piece of new information to be assimilated and stored in the memory it has to be repeated at least seven times.
This means that if we want to learn foreign words, technical terms and definitions, we need to repeat them several times, which requires time input.

Leisure Time

Where can we get additional time for learning?
We can make better use of time otherwise wasted in queues, in transport, or we can study while relaxing on the sofa.
Get this time with the help of our mobile phones.
